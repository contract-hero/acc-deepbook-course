How it runs

A course that runs in your terminal

Install two plugins, run one command. The Agentic Community College framework brings up the sandbox and teaches each lesson section by section, gated by tests that run on a real local chain.

Install

Add the contract-hero marketplace and install the ACC framework plus this course.

Start

Run /acc-deepbook-course:start and pick a lesson from the catalog.

Bootstrap

ACC spins up a local deepbook-sandbox on Docker: real pools, a market maker, a faucet.

Build

Work through each section with an evolving HTML artifact. A live test gate confirms every step.

The Core Four

Four integration patterns, end to end

Each lesson graduates from a runnable reference app and ends green against the sandbox. Take them in order, or jump to the pattern you need.

Lesson 1

Place & Manage Orders

Create a BalanceManager, place and cancel limit and market orders, then reconcile settled against owed balances.

deepBook.placeLimitOrder · cancelAllOrders
Lesson 2

Swap over the CLOB

Swap wallet coins straight over the book with no BalanceManager, guarded by a real minOut that reverts on chain.

deepBook.swapExactQuoteForBase
Lesson 3

Flash-Loan Arbitrage

Borrow with a FlashLoan hot potato and repay it in a single PTB through a Move module you publish on chain.

flashLoans.borrowBaseAsset + Move
Lesson 4

Market-Maker Bot

Quote both sides with a POST_ONLY grid from one BalanceManager, then stake DEEP to earn maker fee rebates.

deepBook.placeLimitOrder · governance.stake
Install

Four lines to your first lesson

Run these inside Claude Code. The course discovers itself from your installed plugins and takes it from there.

claude code
/plugin marketplace add contract-hero/plugin-marketplace
/plugin install agentic-community-college@contract-hero
/plugin install acc-deepbook-course@contract-hero
/acc-deepbook-course:start  # pick a lesson

Before you start

  • ›agentic-community-college@contract-hero ≥ 0.2.0
  • ›Node.js 18+ and pnpm
  • ›Docker running, for the auto-bootstrapped sandbox
  • ›The Sui CLI on your PATH

The course probes each prerequisite before a lesson begins, and offers to clone and bring up the sandbox for you if it is missing.
