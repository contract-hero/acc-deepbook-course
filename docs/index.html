acc-deepbook-course

Build Sui DeepBook integrations inside Claude Code

A hands-on course on the Agentic Community College (ACC) framework. Four lessons take you through the most common DeepBook V3 integration patterns — swaps, orders, flash loans, and market making — each built and tested against a live sandbox.

Install

/plugin marketplace add contract-hero/plugin-marketplace
/plugin install agentic-community-college@contract-hero
/plugin install acc-deepbook-course@contract-hero
/acc-deepbook-course:start

Lessons

  • 02 · Place & Manage Orders — create a BalanceManager, place and cancel limit & market orders, reconcile settled-vs-owed. (Pattern B)
  • 03 · Swap over the CLOB — swap wallet coins with swapExactQuoteForBase; no BalanceManager, with a real minOut slippage guard. (Pattern A)
  • 05 · Flash-Loan Arbitrage — borrow with a FlashLoan hot-potato and repay it in one PTB via a Move module. (Pattern E)
  • 06 · Market-Maker Bot — a two-sided POST_ONLY quote grid from one BalanceManager, staking DEEP for rebates. (Pattern F)
Built on agentic-community-college. Reference sandbox: MystenLabs/deepbook-sandbox.
