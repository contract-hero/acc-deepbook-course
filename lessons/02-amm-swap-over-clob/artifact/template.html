The whole swap, end to end

Wallet coins in, wallet coins out — one transaction, no BalanceManager. DeepBook is the matching engine; you just call it.

1Connect — manifest to a DeepBook client

Every package ID, coin type, and pool lives in the sandbox's deployment manifest. Turn it into a client that knows where DeepBook is.

Key moment: .$extend(deepbook(…)) is what grafts client.deepbook.deepBook.* onto the base client — and its config is derived from the manifest. Get the coin scalar wrong (DEEP 1e6, SUI 1e9) and every amount is off by orders of magnitude.

2Fund & execute

A swap writes to the chain — you need a funded signer and a send that returns enough to see what happened.

Key moment: requesting balanceChanges: true here is load-bearing — Section 4 reads the real fill out of it. Don't ask for it now and the data simply isn't in the response.

3The swap, and the coins you must hand back

swapExactQuoteForBase spends quote (SUI), returns base (DEEP) — straight from wallet coins. The builder hands you back three loose coins.

Key moment: the thunk returns [baseCoin, quoteCoin, deepCoin]. If you don't transferObjects them back to the sender, Move has nowhere to put the value and the whole transaction aborts. One line, most common first-swap bug.

4Read the real fill

How much DEEP did you actually get? Don't guess from the book — read the positive delta of the base coin credited to you in balanceChanges.

Key moment: compare coin types through normalizeStructTag (raw strings won't match the chain's padding → silent 0), keep only positive deltas to your address, then divide by the scalar. baseOut is a genuine on-chain figure.

5Drive it, and prove the slippage guard

Wire the core into the smoke driver and run the live suite — including the test that proves minOut is real on-chain protection.

Key moment: matching MoveAbort code 12 (not just "it threw") proves the guard fired in the pool, on-chain. A network blip would also throw — only the real slippage check produces code 12. That's the lesson: minOut protects you at the matching engine.
